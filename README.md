## 🎯 Суть работы
Целью работы является ручная реализация фундаментальных алгоритмов вычислительной линейной алгебры для комплексного анализа топологии и стабильности взвешенной ориентированной компьютерной сети, состоящей из 6 серверов (S1–S6).

Проект разделён на две логические части:
1. **Математическое ядро (Часть I):** Написание базовых численных методов «с нуля» без использования высокоуровневых библиотечных аналогов.
2. **Прикладной анализ (Часть II):** Интерпретация полученных математических объектов (спектра, норм, собственных векторов) для оценки уязвимости и отказоустойчивости сетевой инфраструктуры.

---

## 🛠️ Что реализовано в коде (по разделам)

* **Раздел 01: Метрики матрицы** — Ручное вычисление столбцовой ($||A||_1$), строчной ($||A||_{\infty}$) и евклидовой ($||A||_F$) норм со сверкой с `numpy`. Расчёт сингулярных чисел (SVD) и вычисление числа обусловленности $\kappa(A)$ для оценки стабильности системы.
* **Раздел 02: Степенной метод** — Итерационный алгоритм поиска доминирующего собственного числа ($\lambda_{max}$) и вектора нагрузок через частное Рэлея. Определение главного генератора трафика в сети.
* **Раздел 03: Обратная итерация со сдвигом** — Метод точечного поиска собственных чисел, ближайших к заданному параметру $\mu$, путём решения систем линейных уравнений.
* **Раздел 04: Процесс Грама–Шмидта** — Алгоритм ортонормализации базиса из собственных векторов матрицы и проверка их взаимной перпендикулярности.
* **Раздел 05: QR-алгоритм** — Реализация классического QR-разложения и QR-алгоритма Шура для одновременного нахождения всего спектра (всех собственных чисел) несимметричной матрицы.
* **Раздел 06: Построение Лапласиана** — Симметризация карты сети и расчёт матрицы Лапласа графа ($L = D - A_{sym}$) для исследования физической связности структуры.
* **Раздел 07: Вектор Фидлера** — Поиск алгебраической связности ($\lambda_2$) и вектора Фидлера методом модифицированной обратной итерации с принудительной ортогонализацией к тривиальному единичному вектору. Логическое разбиение серверов на два изолированных кластера.
* **Раздел 08: Имитация DoS-атаки** — Математическое моделирование падения критического узла сети под нагрузкой и фиксация полного обрушения связности системы до уровня $0.0000$.

## 📊 Результаты визуализации
Программа генерирует интерактивное окно `matplotlib` с 4 графиками:
1. Долгосрочное распределение нагрузки по серверам (подсветка критического узла).
2. Затухающие колебания («пила») на графике сходимости степенного метода из-за наличия отрицательного корня в спектре.
3. Логарифмическое сравнение линейной скорости степенного метода и суперлинейной скорости обратной итерации.
4. Распределение компонент вектора Фидлера (цветовое разделение сети на кластеры раскола).
